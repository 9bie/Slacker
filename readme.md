# BIE插件

- 扫描分析
	- 杀毒检测
	- 检测管理员进程
- 权限维持
	- 服务马
	- winrm后门
	- msdtc劫持
	- 令牌提权
	- 令牌降权
- 小工具
	- 关闭防火墙
	- 弹窗
	- 开启RDP
	- API添加用户
- 提权
	- SweetPotato
	- UAC
	- MS16-032
- 域
	- ZeroLogon
	- PowerView
	- sAMSpoofing
- Dump操作
	- Reg导出
	- comsvcsdll
	- minidump
	- dcsyncdump
