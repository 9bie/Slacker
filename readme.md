# BIE插件
从各种地方搜集来的奇怪的插件

# 功能

- 扫描分析
	- 杀毒检测-进程比对
	- 杀毒检测-WMIC
	- 检测管理员进程
	- 检测硬件信息
	- 检测Domain信息
	- 查询安装的软件
- 权限维持
	- 服务马
	- winrm后门
	- msdtc劫持
	- 令牌提权
	- 令牌降权
- 小工具
	- 关闭防火墙
	- 弹窗
	- 开启RDP
	- API添加用户
	- Defender 加白名单
	- 驱动K进程
- 提权
	- SweetPotato
	- UAC
	- MS16-032
- 域
	- ZeroLogon
	- PowerView
	- sAMSpoofing
	- noPAC
- Dump操作
	- Reg导出
	- comsvcsdll
	- minidump
	- dcsyncdump
	- 浏览器密码
